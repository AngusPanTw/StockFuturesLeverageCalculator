# 股票及期貨庫存槓桿比例計算機 (Stock & Futures Leverage Calculator)

這是一個基於 WPF (.NET 8.0) 開發的桌面應用程式，專為同時持有現股、融資與期貨部位的投資人設計。它可以協助您即時計算投資組合的總曝險、總淨資產以及槓桿倍數。

## ✨ 功能特色

*   **資產整合管理**：
    *   整合銀行存款、股票交割款、期貨權益金。
    *   自動計算總淨資產。
*   **現股庫存管理**：
    *   輸入股票名稱、股數、進場均價、現價。
    *   市值自動計算（現價 × 股數）。
    *   即時損益計算與報酬率 (%)。
    *   紅綠變色顯示：獲利顯示紅色，虧損顯示綠色（台股慣例）。
*   **融資庫存管理**：
    *   支援自訂融資成數（預設融資六成、自備四成）。
    *   自動計算自備金額。
    *   報酬率以自備款為基底，反映實際資金報酬。
    *   融資損益獨立統計，與現股分開顯示。
*   **期貨庫存管理**：
    *   支援 **大型合約 (2000股)** 與 **小型合約 (100股)**。
    *   支援 **多單 (Long)** 與 **空單 (Short)** 部位。
    *   自動計算 **曝險市值**（合約價值）。
    *   即時損益與報酬率計算。
*   **風險監控儀表板**：
    *   **股票市值明細**：現股市值 + 融資市值 = 股票總市值。
    *   **總曝險部位（分子）**：股票總市值 + 期貨曝險。
    *   **淨資產（分母）**：現股市值 + (融資自備款 + 融資損益) + 可用資金 + 交割款 + 期貨權益金。
    *   **槓桿倍數**：總曝險 / 淨資產。
*   **一鍵更新收盤價**：
    *   支援 TWSE（上市）與 TPEX（上櫃）股票收盤價自動查詢。
    *   支援 TAIFEX（期交所）股票期貨收盤價自動查詢。
    *   點擊「更新所有收盤價」按鈕，自動更新所有已填代號的庫存現價。
    *   更新完成後顯示成功/失敗筆數與資料日期。
*   **使用者體驗優化**：
    *   自動儲存投資組合資料（`portfolio.json`），下次開啟自動載入。
    *   數值自動格式化（千分位）。
    *   現股與融資庫存上下分區顯示，各自獨立損益統計。

## 🛠️ 技術堆疊

*   **框架**: .NET 8.0 (WPF)
*   **架構**: MVVM (Model-View-ViewModel)
*   **語言**: C#
*   **資料儲存**: JSON (System.Text.Json)
*   **外部依賴**: 無（純 .NET 內建函式庫）

## 🚀 如何執行

1.  確保您的電腦已安裝 [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)。
2.  使用 Visual Studio 2022 或 VS Code 開啟專案資料夾。
3.  建置並執行：
    ```bash
    dotnet build
    dotnet run
    ```
4.  發布單一執行檔：
    ```bash
    dotnet publish -r win-x64 -c Release --self-contained -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -p:EnableCompressionInSingleFile=true -o ./PublishOutput
    ```

## 📝 使用說明

1.  **輸入資金**：在左上角輸入您的銀行現金、交割款與期貨權益金。
2.  **新增現股**：選擇「現股」，輸入股票名稱、股數、進場均價、現價，點擊新增。
3.  **新增融資**：選擇「融資」，輸入股票資料與融資成數（預設 0.6），點擊新增。
4.  **新增期貨**：輸入代號、口數、方向、成本與市價，選擇合約類型（大/小股票期貨），點擊新增。
5.  **查看報告**：右側面板會即時更新股票市值明細與槓桿倍數。
6.  **刪除項目**：點擊列表右側的「刪除」按鈕即可移除項目。
7.  **更新收盤價**：在庫存中填入股票代號（如 `2330`）或期貨代碼（如 `DIF202603`），點擊「更新所有收盤價」按鈕即可自動更新現價。

## 📡 自動抓取收盤價

### 功能說明

透過官方 API 一鍵更新所有庫存的最新收盤價，取代手動輸入現價。

*   **股票（TWSE/TPEX）**：自動判別上市或上櫃，先查 TWSE，無資料再查 TPEX。
*   **股票期貨（TAIFEX）**：一次取得全部期貨行情，批次比對更新。

### 使用方式

1.  在新增股票時填入「股票代號」（如 `2330`、`2317`）。
2.  在新增期貨時填入「期貨代碼」（格式為 Contract + 月份，如 `DIF202603`；也支援短格式 `DIF2603`）。
3.  點擊左上方的「更新所有收盤價」按鈕。
4.  更新完成後，下方會顯示更新結果與資料日期。

### 注意事項

*   此功能抓取的是**盤後收盤價**，盤中查詢會取得前一交易日的收盤價。
*   TWSE API 有頻率限制（約 3 次/5 秒），多檔查詢時程式會自動加入延遲。
*   代號欄位為空的庫存會被自動跳過，不影響其他庫存的更新。
*   使用者仍可手動輸入或修改現價，不受自動更新影響。
