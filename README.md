# 股票期貨庫存追蹤器

基於 WPF (.NET 8.0) 的桌面應用程式，用於追蹤台灣股票期貨庫存的市值與族群分佈。

> **分支說明**：此為 `feature/2317forever-futures` 分支，功能獨立於 `main` 分支的槓桿計算機。此分支僅追蹤市值，不計算損益。

## 功能特色

- **股票期貨市值追蹤**：輸入股票代號 + 大型/小型 + 年月 + 口數 + 多空，自動查詢合約代號
- **自動查詢合約代號**：透過 TAIFEX API 從股票代號反查期貨合約代碼
- **批次匯入**：以自訂文字格式一次匯入多檔期貨（支援切換大小型、設定年月）
- **一鍵更新收盤價**：批次更新所有期貨的最新收盤價，顯示資料日期與更新結果
- **市值以「萬」為單位**：整數萬不帶小數（`1,234萬`），有餘數顯示一位小數（`345.5萬`）
- **自定義分群**：建立族群（如「記憶體」、「AI」），透過拖曳或下拉選單分配
- **族群市值摘要**：每個群組顯示總市值與總口數
- **快速輸入**：支援 Enter 鍵快速新增期貨與群組
- **自動持久化**：關閉時自動儲存，啟動時自動載入

## 技術堆疊

- **框架**: .NET 8.0 (WPF)
- **架構**: MVVM (Model-View-ViewModel)
- **語言**: C#
- **資料儲存**: JSON (`portfolio.json`)
- **外部依賴**: 無

## 使用說明

### 新增期貨
1. 在左欄輸入**股票代號**（如 `2330`）
2. 從下拉選單選擇**年份**與**月份**
3. 輸入**口數**，選擇**大型/小型**合約與**多/空**方向
4. 按 **Enter** 或點擊「新增股票期貨」按鈕
5. 系統自動查詢 TAIFEX API 取得合約代號與名稱

### 批次匯入
1. 點擊「批次匯入」按鈕開啟匯入視窗
2. 依格式輸入多筆資料：
   ```
   大                ← 切換為大型合約（預設）
   小                ← 切換為小型合約
   26年03月          ← 設定合約年月
   2330 5            ← 股票代號 口數
   2454 3
   ```
3. 點擊「匯入」，系統逐筆查詢並新增

### 分群管理
1. 在右欄輸入群組名稱，按 **Enter** 或點擊「新增群組」
2. 從左側 DataGrid **拖曳**至群組卡片，或在未分群區域使用**下拉選單**選擇目標群組
3. 群組卡片內點擊 X 可移出群組，群組標題旁 X 可刪除整個群組

### 更新收盤價
點擊「更新所有收盤價」按鈕，自動查詢 TAIFEX 期貨行情並更新所有市價。更新完成後顯示資料日期與成功/失敗筆數。

## 注意事項

- 收盤價為盤後資料，盤中查詢會取得前一交易日的收盤價
- 合約代號查詢失敗時會顯示錯誤訊息
- 分群為互斥模式：一檔期貨只能屬於一個群組
- 批次匯入的方向預設全部為多單
