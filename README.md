# 股票及期貨庫存槓桿比例計算機

專為同時持有現股、融資與期貨部位的投資人設計的桌面工具，即時計算總曝險、淨資產與槓桿倍數。

## 功能特色

- **資產整合管理**：整合銀行存款、股票交割款、期貨權益金，自動計算總淨資產。
- **現股庫存**：輸入代號、股數、進場均價，自動計算市值與損益。獲利紅色、虧損綠色（台股慣例）。
- **融資庫存**：支援自訂融資成數（預設六成），報酬率以自備款為基底，與現股獨立統計。
- **期貨庫存**：支援大型合約（2000股/口）與小型合約（100股/口），支援多空方向，輸入股票代號後自動查詢合約代號與名稱。
- **槓桿儀表板**：即時顯示股票市值、期貨曝險、總曝險、淨資產與槓桿倍數。
- **一鍵更新收盤價**：自動從證交所/櫃買中心/期交所抓取盤後收盤價，支援上市、上櫃股票與股票期貨。
- **批次匯入**：股票與期貨皆支援貼上多筆資料一次匯入，自動查詢名稱，部分失敗不影響成功項目。
- **自動儲存**：關閉程式時自動儲存，下次開啟自動載入。

## 槓桿公式

```
總曝險 = 股票總市值 + 期貨曝險
淨資產 = 現股市值 + (融資自備款 + 融資損益) + 可用資金 + 交割款 + 期貨權益金
槓桿倍數 = 總曝險 / 淨資產
```

## 如何執行

需安裝 [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)。

```bash
dotnet build    # 建置
dotnet run      # 執行
```

發布單一執行檔：

```bash
dotnet publish -r win-x64 -c Release --self-contained -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -p:EnableCompressionInSingleFile=true -o ./PublishOutput
```

## 使用說明

1. **輸入資金**：在左上角輸入銀行現金、交割款與期貨權益金。
2. **新增股票**：選擇現股或融資，輸入代號、股數、進場均價，點擊新增。
3. **新增期貨**：輸入股票代號、選擇年月與合約類型（大/小），輸入口數與成本價，點擊新增。合約代號與名稱會自動填入。
4. **批次匯入股票**：點擊「批次匯入股票」，貼上多筆資料（每行：`代號 股數 進場價`）。
5. **批次匯入期貨**：點擊「批次匯入期貨」，選擇年月後貼上資料（每行：`代號 口數 成本價`）。
6. **更新收盤價**：點擊「更新所有收盤價」，自動抓取最新盤後收盤價。
7. **查看報告**：右側面板即時顯示市值明細與槓桿倍數。
8. **刪除項目**：點擊列表中的「刪除」按鈕。

### 收盤價注意事項

- 抓取的是**盤後收盤價**，盤中查詢會取得前一交易日資料。
- 代號欄位為空的庫存會自動跳過。
- 期貨若缺少合約代號，更新時會自動補齊。
- 仍可手動輸入或修改現價，不受自動更新影響。
