# 股票庫存槓桿比例計算機 (Stock Leverage Calculator)

這是一個基於 WPF (.NET 8.0) 開發的桌面應用程式，專為持有現股與融資部位的投資人設計。可即時計算投資組合的總曝險、淨資產與槓桿倍數，並支援一鍵更新收盤價。

## 功能特色

- **股票庫存管理**：
    - 區分現股與融資兩種持有類型
    - 記錄股票代號、名稱、股數、進場均價、現價
    - 自動計算總市值、損益、報酬率、融資自備款
    - 損益紅綠變色顯示（獲利紅色、虧損綠色，台股慣例）
- **自動查詢收盤價**：
    - 一鍵更新所有股票收盤價（TWSE 上市 / TPEX 上櫃）
    - 查詢失敗自動重試（漸進式延遲）
- **批次匯入**：
    - 現股/融資分區文字輸入，每行格式 `代號 股數 進場均價`
    - 自動查詢股票名稱（透過 TAIFEX API 快取）
    - 支援部分成功匯入
- **風險監控儀表板**：
    - 總曝險 = 股票總市值
    - 淨資產 = 現股市值 + 融資自備款 + 融資損益 + 可用資金 + 交割款
    - 槓桿倍數 = 總曝險 / 淨資產
- **使用者體驗**：
    - 自動儲存投資組合（`portfolio.json`），重開自動載入
    - DataGrid 欄寬自動調整
    - 啟動時預載 TAIFEX 快取，加速首次股票名稱查詢

## 技術堆疊

- **框架**: .NET 8.0 (WPF)
- **架構**: MVVM (Model-View-ViewModel)
- **語言**: C#
- **資料儲存**: JSON (System.Text.Json)
- **外部依賴**: 無（僅使用內建 .NET 函式庫）

## 如何執行

```bash
dotnet build
dotnet run
```

或發布為單一執行檔（win-x64）：

```bash
dotnet publish -r win-x64 -c Release --self-contained -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -p:EnableCompressionInSingleFile=true -o ./PublishOutput
```

## 使用說明

1. **輸入資金**：在上方輸入可用現金與交割款
2. **新增股票**：填入代號、名稱、類型（現股/融資）、股數、均價、現價，點擊新增
3. **批次匯入**：點擊「批次匯入股票」，在現股/融資區塊貼上多筆資料
4. **更新收盤價**：點擊「更新所有收盤價」，自動從 TWSE/TPEX 抓取最新收盤價
5. **查看報告**：右側面板即時顯示市值明細、槓桿倍數
6. **刪除項目**：點擊列表右側的 X 按鈕移除
